Welcome

Please enter a valid bip39 mnemonic. This page does not generate them randomly; if you wish to create a new mnemonic, please use a wallet you trust.

Some bip39 wallets support the BIP39 plausible deniability passphrase. It affects the wallet identity and produces different results. There are no restricitons on length or content.

BIP44 structured wallets support multiple accounts. Each account will have a different GPG identity.

If you enter a long enough password here the generate button will output a gpg private key instead of only the public key part. This takes much longer as the gpg key must be s2k encrypted with the password you choose. This page wont accept less than 75 bytes of entropy in an s2k password.
